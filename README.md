# Spatial Equity Analysis for STEAM Teachers in Uganda

This repository contains the code and data used to analyze the spatial distribution and equity of secondary STEAM teachers across districts in Uganda.

The analysis was conducted as part of a research study on teacher distribution and educational equity in Uganda.

## Repository Contents

- `plots_9_panels.ipynb` : Main Python code to generate 9-panel spatial figure (a-i). *Will be uploaded prior to final publication.*
- `NEW PYTHON CODE.docx` : Python code used for the spatial analysis
- `SECONDARY ONLY.xlsx` : District-level dataset containing student and teacher numbers used in the analysis
- `NEW PLOTS.docx` : Output figures generated from the analysis

## How to Reproduce the Results

1. Open `plots_9_panels.ipynb` in Google Colab or Jupyter Notebook
2. Upload `SECONDARY ONLY.xlsx` to the Colab environment
3. Run all cells to reproduce the 9-panel figure (a-i) shown in the paper

## Requirements
- Python 3.9+
- Libraries: pandas, geopandas, matplotlib, seaborn
